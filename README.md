# sedgewick-wayne
Creative Exercises from Computer Science: An Interdisciplinary Approach by
Robert Sedgewick and Kevin Wayne.

_NOTE_: We depend upon the book's standard library: [stdlib.jar](https://introcs.cs.princeton.edu/java/stdlib/stdlib-package.jar) from [this page](https://introcs.cs.princeton.edu/java/stdlib/).  We download this into the /lib/ directory (not checked in).
